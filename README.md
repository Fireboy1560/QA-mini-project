 QA Mini Project – SauceDemo

 Project Objective
Manual testing of the SauceDemo (Swag Labs) demo site for a Junior QA position.

 What does it include?
- 5 test cases (Excel) – positive and negative scenarios for login and the shopping cart
- 3 bug reports (Jira) – with actually reproducible defects
- List of edge cases and test ideas
